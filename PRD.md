## 1. 產品概述

### 產品名稱
AI Tarot - 智能塔羅牌

### 產品類型
PWA Web 應用 (可安裝至主螢幕)

### 核心功能
AI 驅動的塔羅牌占卜，提供即時、私密、價格親民的心靈指引

### 目標用戶
- 18-45 歲尋求靈性指引的成年人
- 對塔羅牌有興趣但未接觸過的初學者
- 尋求快速、私密解答的用戶

## 3. 用戶介面需求

### 3.1 頁面結構

```
首頁 (Home)
├── 牌陣選擇 (Spread Selection)
├── 抽牌區 (Card Area)
├── 解讀結果 (Reading Result)
├── 歷史記錄 (History)
├── 設定 (Settings)
└── 關於 (About)
```

### 3.2 設計系統

#### 顏色
| 用途 | 顏色 |
|------|------|
| 主色 | #6366f1 (Indigo) |
| 深色 | #4f46e5 |
| 點綴 | #f59e0b (Amber) |
| 背景 | #0f0f1a (Dark) |
| 卡片 | #1a1a2e |
| 文字 | #f8fafc |
| 文字次要 | #94a3b8 |

#### 字體
- 標題: Cinzel (神秘感)
- 正文: Noto Sans TC (中文支援)

#### 動畫
- 卡片翻轉: 3D transform
- 抽牌動畫: 滑入 + 放大
- 結果顯示: 淡入 + 上滑

### 3.3 響應式設計

| 斷點 | 尺寸 | 布局 |
|------|------|------|
| Mobile | < 640px | 單列，底部導航 |
| Tablet | 640-1024 | 雙列 |
| Desktop | > 1024 | 居中最大寬度 |

## 4. 技術需求

### 4.1 技術棧
- 前端: HTML5, CSS3, Vanilla JavaScript
- 存儲: LocalStorage (PWA)
- 圖片: SVG + Data URI
- 構建: Vite/無需構建

### 4.2 PWA 要求
- ✅ Service Worker (離線支持)
- ✅ Web App Manifest
- ✅ 可添加到主螢幕
- ✅ 離線可用

### 4.3 性能目標
- 首屏載入: < 2秒
- 交互反饋: < 100ms
- 離線可用: 完全
